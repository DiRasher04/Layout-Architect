2D Top-Down RPG с процедурной генерацией уровней, написанный на Godot 4. Проект сочетает классический геймплей dungeon crawler'а с гибридными алгоритмами генерации подземелий.

✨ Особенности
🎮 Геймплей
Управление: WASD — движение, Shift — ускорение, Стрелки — атака

Боевая система: Атака мечом с анимацией взмаха по дуге (85°)

Враги: Преследование, рывок к игроку, система задержки перед атакой

Смерть и воскрешение: Анимация смерти → 3 сек мёртвое тело → рестарт уровня

🏗️ Процедурная генерация (3 алгоритма)
Проект реализует три подхода к генерации уровней:

BSP	- Рекурсивное разбиение пространства на комнаты + коридоры	Структурированные подземелья
Cellular Automata -	Клеточный автомат с шумом 45% и сглаживанием	Органические пещеры
Гибридный	- BSP для структуры + Cellular для деталей	Лучшее из обоих миров
🧱 Автотайлинг
Terrain Set 0: Пол (автосоединение)

Terrain Set 1, Set 0: Стены (с автотайлингом)

Terrain Set 1, Set 1: Фундамент под стенами (отдельный слой)

🛠️ Технический стек
Движок: Godot 4.3+

Язык: GDScript

Графика: Пиксель-арт (2D)
